📍 WHERE TO PUT YOUR API KEYS?
==============================

File: .env
Location: Your project root directory

Example paths:
  Mac:    /Users/YourName/Documents/edoska/.env
  Linux:  /home/user/edoska/.env
  Windows: C:\Users\YourName\Desktop\edoska\.env

How to find it:
  1. Open your project folder
  2. Look for file named ".env"
  3. Open with any text editor


🔑 YOUR CURRENT .env FILE
=========================

Currently has these placeholders:

VITE_YOUTUBE_API_KEY=your_youtube_api_key_here
VITE_OPENAI_API_KEY=your_openai_api_key_here
VITE_GEMINI_API_KEY=your_gemini_api_key_here
VITE_TELEGRAM_BOT_TOKEN=your_telegram_bot_token_here
VITE_API_URL=http://localhost:3001


📝 3 STEPS TO ADD YOUR KEYS
===========================

STEP 1: Get your API keys from these websites:
  1. YouTube API: https://console.cloud.google.com/
  2. OpenAI API: https://platform.openai.com/api-keys
  3. Gemini API: https://aistudio.google.com/app/apikey
  4. Telegram: Chat with @BotFather on Telegram

STEP 2: Open .env file and replace the placeholders:
  Before: VITE_YOUTUBE_API_KEY=your_youtube_api_key_here
  After:  VITE_YOUTUBE_API_KEY=AIzaSyD1234567890

STEP 3: Save the file and restart dev server:
  npm run dev

✅ HOW TO TEST
==============

After adding a key:

1. Restart dev server (npm run dev)
2. Open browser (http://localhost:5173)
3. Press F12 to open Developer Console
4. Type: console.log(import.meta.env.VITE_YOUTUBE_API_KEY)
5. You should see your actual key

If you see "undefined":
  → Key not loaded
  → Restart server: npm run dev
  → Try again
