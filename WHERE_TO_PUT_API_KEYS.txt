🔧 HOW TO ADD YOUR API KEYS (Simple Steps)
===========================================

STEP 1: Open the .env file
  • Navigate to your project folder
  • Find the file named ".env"
  • Right-click → Open with → Text Editor (or VS Code)

STEP 2: Replace the placeholders with your real keys
  
  Example for YouTube API:
  ┌─────────────────────────────────────────────────────┐
  │ BEFORE:                                             │
  │ VITE_YOUTUBE_API_KEY=your_youtube_api_key_here     │
  │                                                     │
  │ AFTER:                                              │
  │ VITE_YOUTUBE_API_KEY=AIzaSyD1234567890abcdef       │
  └─────────────────────────────────────────────────────┘
  
  Example for OpenAI API:
  ┌─────────────────────────────────────────────────────┐
  │ BEFORE:                                             │
  │ VITE_OPENAI_API_KEY=your_openai_api_key_here       │
  │                                                     │
  │ AFTER:                                              │
  │ VITE_OPENAI_API_KEY=sk-proj-abc123xyz              │
  └─────────────────────────────────────────────────────┘

STEP 3: Save the file
  • Press Ctrl+S (or Cmd+S on Mac)
  • Close the editor

STEP 4: Restart the dev server
  • In terminal, press Ctrl+C to stop current server
  • Run: npm run dev
  • Done! ✅


🔑 HOW TO GET YOUR API KEYS
===========================

YOUTUBE API KEY:
  1. Go to: https://console.cloud.google.com/
  2. Create a new project
  3. Search for "YouTube Data API v3" and enable it
  4. Go to Credentials → Create API Key
  5. Copy the key and paste in .env

OPENAI API KEY:
  1. Go to: https://platform.openai.com/api-keys
  2. Sign up / Log in
  3. Click "Create new secret key"
  4. Copy the key (save securely!)
  5. Paste in .env

GOOGLE GEMINI API KEY:
  1. Go to: https://aistudio.google.com/app/apikey
  2. Click "Get API key"
  3. Create API key in Google Cloud
  4. Copy and paste in .env

TELEGRAM BOT TOKEN:
  1. Open Telegram
  2. Chat with @BotFather
  3. Send: /newbot
  4. Follow the prompts
  5. Copy token to .env

❓ FAQ - QUICK ANSWERS
======================

Q: Do I NEED to add API keys?
A: NO! App works fine with mock data. Add keys when you want real power.

Q: Where exactly is the .env file?
A: In your project root folder. Same level as package.json, src/, dist/

Q: How do I open .env?
A: Right-click on it → Open with → Text Editor (or drag to VS Code)

Q: Can I use just one API key?
A: Yes! Add YouTube only, or OpenAI only. All optional.

Q: What if I add a wrong key?
A: It won't work. Check browser console for errors.

Q: Do I need to commit .env to Git?
A: NO! It's in .gitignore. Never commit it.

Q: Can I have different keys for dev/prod?
A: Yes! Create .env.local for dev, .env.production for prod.

Q: Will my keys be exposed?
A: No! Variables with VITE_ are for frontend only. Secret keys stay in backend.

Q: Can I test without restarting?
A: Usually no. Restart with: npm run dev
