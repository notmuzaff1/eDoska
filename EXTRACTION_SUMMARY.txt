================================================================================
JSON STRUCTURE
================================================================================

Each lesson contains:

{
  "dayOfWeek": "Monday|Tuesday|Wednesday|Thursday|Friday|Saturday",
  "timeStart": "HH:MM",        // 24-hour format, e.g., "08:00"
  "timeEnd": "HH:MM",          // 24-hour format, e.g., "08:45"
  "subject": "Subject Name",   // Full subject name in Uzbek
  "teacher": "Full Name",      // Teacher's full name
  "room": "Room Number",       // Classroom or location
  "className": "10-V"          // Always "10-V" for this schedule
}

Example:
{
  "dayOfWeek": "Monday",
  "timeStart": "08:00",
  "timeEnd": "08:45",
  "subject": "Ingliz tili Maxsus",
  "teacher": "Farxod Karimov",
  "room": "101",
  "className": "10-V"
}
