## 📖 For Developers

### Understanding the Code

**Main Flow:**
1. Open `src/App.tsx` - Entry point
2. Check `src/components/Dashboard.tsx` - Main layout
3. See `src/lib/scheduleManager.ts` - Lesson detection
4. Review `src/components/StudentPicker.tsx` - Example component

### Key Files

- **Schedule Logic:** `src/lib/scheduleManager.ts`
- **Real Schedule:** `src/data/schedule.json`
- **Real Students:** `src/data/students.json`
- **AI Generation:** `src/lib/aiLessonGenerator.ts`
- **Styling:** `tailwind.config.js`

### Commands

```bash
npm run dev        # Start dev server
npm run build      # Create production build
npm run lint       # Check code quality
npm run typecheck  # TypeScript verification
npm run preview    # Test production build
```
