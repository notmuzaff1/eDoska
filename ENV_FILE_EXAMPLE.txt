🔧 HOW TO ADD YOUR API KEYS
============================

1. Open the .env file with any text editor (VS Code, Notepad, etc.)

2. Replace "your_youtube_api_key_here" with your actual YouTube API key:
   ❌ VITE_YOUTUBE_API_KEY=your_youtube_api_key_here
   ✅ VITE_YOUTUBE_API_KEY=AIzaSyD1234567890abcdefg

3. Replace "your_openai_api_key_here" with your actual OpenAI API key:
   ❌ VITE_OPENAI_API_KEY=your_openai_api_key_here
   ✅ VITE_OPENAI_API_KEY=sk-proj-123456789abcdefg

4. Save the file

5. Restart the dev server:
   - Press Ctrl+C to stop current server
   - Run: npm run dev


🔑 GETTING YOUR API KEYS
========================

YouTube API Key:
  👉 Go to: https://console.cloud.google.com/
  1. Create new project
  2. Enable "YouTube Data API v3"
  3. Create API Key in Credentials
  4. Copy the key

OpenAI API Key:
  👉 Go to: https://platform.openai.com/api-keys
  1. Sign up / Log in
  2. Create new secret key
  3. Copy it (you won't see it again!)

Google Gemini API Key:
  👉 Go to: https://aistudio.google.com/app/apikey
  1. Create API key
  2. Copy it

Telegram Bot Token:
  👉 Open Telegram and chat: @BotFather
  1. Send /newbot
  2. Follow prompts
  3. Copy your token


✅ TESTING YOUR SETUP
====================

After adding keys and restarting server:

1. Open your browser (http://localhost:5173)
2. Open Developer Console (Press F12)
3. In Console tab, type:
   console.log(import.meta.env.VITE_YOUTUBE_API_KEY)

4. If you see your key → ✅ It works!
   If you see "undefined" → ❌ Key not loaded (restart server)

🔒 SECURITY NOTE
================

✅ DO:
  • Keep .env private (it's in .gitignore)
  • Use different keys for dev/production
  • Rotate keys occasionally
  • Set API quotas/limits

❌ DON'T:
  • Commit .env to Git
  • Share your .env file
  • Expose secret keys on frontend
  • Use same key for dev and production
